Overview of the Dashboard

The Satellite Performance Monitoring Dashboard is designed to simplify the tracking of satellite health by transforming overwhelming raw data into easily understandable visualizations. This tool provides users with key insights into the performance metrics of satellites, enabling better decision-making and proactive management.

Key Features

  • Data Monitoring: Continuously updates key performance metrics to ensure users have the accurate information.
  • Visualization Tools: Utilizes charts and graphs to present complex data in a visually accessible manner.

About

Project Background

This project was initiated to address the challenge of tracking satellite health amidst overwhelming raw data. By creating a user-friendly dashboard, we aim to provide a solution that visualizes key performance metrics, making it easier for users to monitor and analyze satellite performance.

Team Members

  • Joud Al-Amri: Responsible for handling and organizing the data, ensuring it is accurate and up-to-date.
  • Rayan: Analyzes the data and generates reports, ensuring that key metrics are displayed correctly on the dashboard.
  • Raghad Garout: Develops and programs the interface of the dashboard, focusing on usability and functionality.

Technology Stack

The dashboard is built using the following technologies:

  • SQL: For database management and data retrieval.
  • Excel: For initial data analysis and manipulation.
  • HTML: For structuring the web interface.
  • JavaScript: For interactive elements and dynamic data handling.
  • Chart.js: For creating visualizations of performance metrics.

Industrial Engineering Applications

  • Reliability Engineering: The dashboard aids in predicting potential satellite failures, allowing for proactive maintenance scheduling.
  • Process Optimization: Analyzing performance trends enables improved decision-making for mission planning and satellite operations.
  • Lean Systems & Waste Reduction: By minimizing unexpected failures, we reduce downtime and unnecessary maintenance costs.

Reports

From our analysis, we found that the majority of satellites in the dataset were launched for communication and Earth observation purposes. Low Earth Orbit (LEO) was by far the most common orbit type. We also noticed a wide range in satellite power levels, which seems to vary depending on the mission type and orbit class. These kinds of observations, even from static data, help show the importance of performance monitoring and data visualization in aerospace and satellite operations.
